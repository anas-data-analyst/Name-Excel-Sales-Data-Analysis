#  📊 Sales Data analysis Project (Excel)
###📌 Project Overview 
this project showcases a complete sales analysis dashboard built using Microsoft Excel. It focus on regional performance and category-wise sales distribution.
---
###👉 Featurs & Tools:
-**Data Summarization:**used  Pivot Tables to aggregate sales data.
-**visualization:**Created interactive pIE Charts for quick insights.
-**Interactivit:** Added Slicers to filter data by region and category.
